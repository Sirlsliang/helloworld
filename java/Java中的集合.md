## Java中的集合

### Collection
  - 一组对象的集合，一般称为元素，不会提供具体的实现，为子接口提供实现
  - 所有集合实现最少应该提供两个构造函数，一个是空参数的构造函数，一个是需要一个集合作为参数的构造函数。
  ![集合的方法](collection.png 集合方法)
  
### Iterator(迭代器)
  - 集合框架中使用迭代器来代替枚举
  - 迭代器可以删除正在操作的集合的元素

### List
  一个有序的集合，能够精确控制集合中的元素，允许有重复的元素，可以有两个相等的元素，也可以有null元素。
  - ArrayList
  - LinkedList

### Set
  - HashSet
  - LinkedHashSet
